Flex-Direction

Para Conteiner

A configuração de direção automaticamente para o modo "flex-direction: row;" que seria o modo em linha, da esquerda para a direita ja para utilizar a direção da vertical de cima para baixo você usa o "flex-direction: colum;" agora para inverter esta direção você pode utilizar o flex-direction: row ou colum reverse; resumindo se for row é da esquerda para a direita em linha se for colum de de cima para baixo em coluna.

Eixos

Nós temos que ficar ligados no surgimento dos eixos ao criar um Conteiner o main start vai sempre da onde os items estão começando se for normal vai ser da esquerda para a direita então ele começa na esquerda, o cross-start faz uma linha cruzando a main start na horizontal.
